# Sample Datasets

```shell
usage: sample.py [-h] [-n N] [-t [TYPE ...]] input_file

sub-sample data sets

positional arguments:
  input_file            input file

options:
  -h, --help            show this help message and exit
  -n N, --number N      number of samples (default 100)
  -t [TYPE ...], --type [TYPE ...]
                        list of event types to be sampled
```
